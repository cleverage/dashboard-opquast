Opquast dashboard

This dashboard lists the Opquast partner agencies. But above all, it makes it possible to generate pretty tables of data as well as a nice chart of certifieds by levels.

Opquast means Open Quality Standards. The project has been launched in 2004. The goal of the projet is to improve the quality of the Web by writing and publishing simple, checkable, realistic, universal and transversal checklists. The main publication is the Opquast Web quality checklist.
